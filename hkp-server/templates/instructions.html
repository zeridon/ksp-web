What is this

This is a HKP compliant keyserver used for organizing keysigning parties. You can submit your keys and they will be bundled as needed.

How to use it

Send your keys with a normal gpg 
gpg --keyserver <this address> --send-keys 0xA16212118ACB4CC5
Get keys with a normal gpg 
gpg --keyserver <this address> --recv-keys 0xA16212118ACB4CC5
Get all keys in the server 
curl <this server>/all-keys
